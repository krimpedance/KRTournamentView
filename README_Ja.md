`KRTournamentView` は様々な構造に対応できる柔軟なトーナメント表です．

<img src="https://raw.githubusercontent.com/krimpedance/Resources/master/KRTournamentView/styles.png" width="100%" />


## 必要環境
- iOS 8.0+
- Xcode 11.0+
- Swift 5.0+


## デモ
`DEMO/`以下にあるサンプルプロジェクトから確認してください．

または，[Appetize.io](https://appetize.io/app/yevb2mx7ea7p10cjqb9fp3tenm?device=iphonex&scale=75&orientation=portrait&osVersion=13.3)にてシュミレートしてください．


## インストール
いくつかの方法で KRTournamentView をインストールすることができます．

+ [Swift Package Manager](https://swift.org/package-manager)

`File` -> `Swift Packages` -> `Add Package Dependency` on Xcode.app

+ [CocoaPods](http://cocoapods.org)

```ruby
# Podfile
pod "KRTournamentView"
```

+ [Carthage](https://github.com/Carthage/Carthage)

```ruby
# Cartfile
github "Krimpedance/KRTournamentView"
```
